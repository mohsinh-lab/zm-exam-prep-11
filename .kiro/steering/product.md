# Product Overview

AcePrep 11+ is a smart exam preparation Progressive Web App (PWA) designed for 9-12 year olds preparing for the UK 11+ GL Assessment to get into their dream school.

## Core Features

- Interactive adaptive learning with booster missions that adjust to student progress
- Multi-subject coverage (Maths, English, Verbal Reasoning, Non-Verbal Reasoning)
- Performance tracking with XP, gems, ranks, and visual progress monitoring
- Parent portal for monitoring student progress
- Themed graphics (Pokemon/Transformers) and motivational Islamic quotes
- Cloud sync with Firebase for cross-device progress persistence
- Installable as PWA on mobile devices (iPad, iPhone, etc.)

## Target Users

- Primary: Students aged 9-12 preparing for 11+ exams
- Secondary: Parents monitoring their child's progress

## Key Dates

- Target exam date: September 15, 2026
- Countdown timer displayed on student home screen
